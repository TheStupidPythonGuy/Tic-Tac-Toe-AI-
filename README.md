# Tic-Tac-Toe-AI-
A 1 player Tic Tac Toe game with AI

## Features

- 1 player
- Fun
- Easy to play

## Requirements

- Python 3.x

## Installation

1. Download the file and run **chmod +x /path/to/Converter.py** in your terminal.
2. Run **./path/to/Converter.py**
